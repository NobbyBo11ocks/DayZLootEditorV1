# DayZ Loot Editor Website v2

## Included
- `index.html` — one-page homepage
- `screenshots/` — local screenshots used by the homepage

## Updated in this version
- darker DayZ-style theme
- GitHub repository section
- GitHub releases download buttons
- creator profile block with GitHub avatar
- screenshot gallery
- validation feed feature highlight

## Links already wired in
- Repo: https://github.com/NobbyBo11ocks/DayZLootEditorV1
- Releases: https://github.com/NobbyBo11ocks/DayZLootEditorV1/releases
- Profile: https://github.com/NobbyBo11ocks

## How to use
1. Upload the whole folder to GitHub Pages, Netlify, or any static host.
2. Keep the `screenshots` folder next to `index.html`.
3. Open the site and test the GitHub download/repo/profile buttons.

## Local preview
Double-click `index.html` or open it in your browser.

What it helps with

Manual DayZ economy editing is easy to mess up. This tool is built to make the workflow faster, safer, and easier to review.

Loot Editor

Open a mission folder or direct types.xml, search fast, edit values cleanly, and save with fewer mistakes.

Custom CE Manager

Create, register, repair, and manage modular CE files without dumping everything into one giant file.

Validation Feed

See clear warnings before save time, like broken count relationships, quantity issues, and CE setup problems.
